# Changelog

All notable changes to the Oncology Dashboard project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-01

### Added
- Initial project setup and documentation
- Professional README with comprehensive documentation
- Git configuration files (.gitignore, .gitattributes)
- Contributing guidelines
- Project structure documentation
- License (MIT)

### Semantic Model
- Core tables: Gold_Medicine_Metrics_Source, Gold_Oncology_Dashboard_Source
- Calendar dimensions for time intelligence
- Optimized relationships and DAX expressions
- Multi-language support (en-US)

### Reports & Visualizations
- 10+ interactive dashboard components
- Geographic mapping with worldtopo.json integration
- Responsive design across multiple screen sizes
- Professional theming with built-in themes

### Infrastructure
- PBIP project format for better version control
- TMDL-based model definition (Infrastructure as Code)
- Semantic model separation from reports
- Production-ready architecture
